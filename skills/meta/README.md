## 架构指南说明

架构指南是**教 AI 如何设计系统**的文档，包含：
- 设计原则
- 代码示例（TypeScript + Python）
- 分步实现指南
- 问题排查

TypeScript 版为主版本（生产验证），Python 版标记为社区贡献待验证。
